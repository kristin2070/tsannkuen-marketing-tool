🔑 設定

🔰 第一次使用？點這裡看 5 步驟申請 Token
1️⃣ 開 GitHub Settings → Tokens (classic)
2️⃣ 點「Generate new token (classic)」
3️⃣ Note 填「Apple Shop 文案器」，Expiration 選 90 days
4️⃣ Scopes 全部不用勾（預設就能用）
5️⃣ 拉到底點綠色「Generate token」
6️⃣ 複製 ghp_xxx 字串貼到上方欄位

🔒 Token 只存在你的瀏覽器，不會上傳，安全無虞
⚠️ 跳 413 太長 / 429 額度不足 → 換另一個模型即可
